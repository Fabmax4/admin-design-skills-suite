# 样式边界

## 色彩

- 品牌主色：`#fa8c16`
- 成功：沿用 `Ant Design` 绿色语义
- 警告：沿用橙 / 金色语义
- 错误：沿用红色语义
- 信息：沿用蓝色语义

规则：

- 主色只用于关键按钮、重点状态、少量强调
- 语义色只表达状态，不承担装饰任务
- 中性色优先承担结构和阅读任务

## 圆角

- 允许档位：`6 / 8 / 12 / 16`
- 信息卡、表单、表格容器优先 `8` 或 `12`
- 不使用大圆角制造“柔软卡片秀”

## 密度

- 左侧导航展开宽度：`176px`
- 以紧凑、可扫读为目标
- 内边距优先统一，不单独发明节奏

规则：

- 通过分组和版式控制密度
- 不通过把所有字都缩小来换空间
