# Quick Start

6 条最短调用覆盖日常后台设计任务。中英文都可以发起；推荐从 `$admin-design-orchestrator` 开始。

```text
$admin-design-orchestrator 设计后台页
$admin-design-orchestrator design admin page
$admin-design-review 审查页面
$admin-design-components 抽组件契约
$admin-design-style 收视觉
$admin-design-visualization 判断是否上图
```

## 选择规则

- 不确定从哪层开始：用 `$admin-design-orchestrator`。
- 已有页面或方案需要验收：用 `$admin-design-review`。
- 要把业务块抽成可复用组件：用 `$admin-design-components`。
- 视觉变得花、散、圆角过重或色彩过多：用 `$admin-design-style`。
- 不确定是否需要图表：用 `$admin-design-visualization`。
- 只需要补状态变化、焦点转移或操作反馈：用 `$admin-design-motion`。
